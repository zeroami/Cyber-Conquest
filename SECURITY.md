# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project's code or documentation, please send an email to the repository owner. All security vulnerabilities will be promptly addressed.

## Responsible Disclosure

We follow a responsible disclosure policy:

1. **Do NOT** open a public issue for security vulnerabilities
2. **DO** contact the maintainer privately
3. Allow reasonable time for the vulnerability to be addressed before public disclosure

## Educational Purpose Warning

This repository contains educational material about cybersecurity techniques. All content is provided for:
- Academic research
- Authorized penetration testing
- Defensive security education
- Ethical hacking training

**Any use of these techniques against systems without explicit written authorization is illegal and unethical.**

## Safe Usage Guidelines

1. Always use isolated lab environments (VMs, containers)
2. Never test against systems you don't own or have written permission to test
3. Follow all applicable laws and regulations
4. Review the [Legal Guide](./08-Gobernanza/Guia_Legal.md) before practicing any technique

## Contact

For security concerns, please open a private issue or contact through GitHub.
